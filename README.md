# ClonaStatus
## API - Usage

• Usage: 

```php
<?php
  include ('StatusApi.php'); //Include php file
  use io\clonalejandro\StatusApi; // Import a StatusApi class

  $req = new StatusApi('clonalejandro.me');

  if ($req->isOnline()) echo "Online";
  else echo "Offline";
?>
```
